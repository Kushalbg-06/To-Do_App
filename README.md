# 📝 To-Do App

A full-stack To-Do application built using **HTML, CSS, JavaScript** for frontend and **FastAPI** for backend.
This app allows users to create and manage tasks easily.

---

## 🚀 Live Demo

* 🌐 Frontend: https://to-do-app-xi-bice.vercel.app/
* 🔗 Backend API: https://todo-backend-f205.onrender.com/

---

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* FastAPI
* SQLite

### Deployment

* Vercel (Frontend)
* Render (Backend)

## 🧪 Run Locally

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
open index.html
```
